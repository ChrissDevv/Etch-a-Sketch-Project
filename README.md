# 🎨 Etch a Sketch

A browser-based pixel drawing app built with HTML, CSS, and JavaScript — inspired by the classic Etch a Sketch toy.

## Live Demo

[View Project](https://chrissdevv.github.io/Etch-a-Sketch-Project/) 

## Features

- 16x16 drawing grid generated dynamically with JavaScript
- Hover to draw — move your mouse across the grid to leave a colored trail
- Reset button that prompts the user for a custom grid size (up to 100x100)
- Grid always fills the same 960px space regardless of size
- Fully responsive square sizing calculated dynamically

## How to Use

1. Hover your mouse over the grid squares to draw
2. Click the **Reset** button to clear the grid
3. Enter a number when prompted to set a new grid size (e.g. 32 for a 32x32 grid)
4. Start drawing again!

## Built With

- HTML5
- CSS3 (Flexbox)
- Vanilla JavaScript (DOM manipulation, event listeners)

- Adding interactivity with `addEventListener` and `mouseover` events
